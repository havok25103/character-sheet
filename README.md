# README

This is a personal tool that is still under development.

## Setup

- **Ruby:** 3.1.0
- **Rails:** 7.0.2.3
- **Bundler:** 2.3.6

1. Clone main
2. `bundle install`
3. `rake db:create`
4. `rake db:migrate`
5. `rake db:seed`
6. `rails s`

**To lint:** `bundle exec rubocop`

**To test:** `bundle exec rspec`
